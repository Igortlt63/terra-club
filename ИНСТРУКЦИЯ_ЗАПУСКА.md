### Подробное руководство (понятно даже ребёнку)

---

## ШАГ 1: Установите Node.js (один раз)

**Что такое Node.js?**
Это программа, которая позволяет запускать веб-приложения на вашем компьютере.

**Как установить:**

1. Откройте браузер и перейдите на сайт: **https://nodejs.org**
2. Вы увидите две кнопки. Нажмите на левую — **"LTS"** (это стабильная версия)
3. Скачается файл-установщик. Запустите его двойным кликом
4. Нажимайте **"Next"** (Далее) во всех окнах, ничего не меняя
5. В конце нажмите **"Finish"** (Готово)

**Проверка установки:**
- На Windows: нажмите клавиши `Win + R`, напишите `cmd`, нажмите Enter
- На Mac: нажмите `Cmd + Пробел`, напишите `terminal`, нажмите Enter
- Введите команду: `node --version`
- Если видите что-то вроде `v20.11.0` — всё готово ✅

---

## ШАГ 2: Распакуйте папку с приложением

1. Скачайте файл `terra-club.zip`
2. Найдите его в папке **Загрузки**
3. Кликните правой кнопкой мыши → **"Извлечь всё"** (Windows) или просто двойной клик (Mac)
4. Выберите место, куда хотите распаковать (например, **Рабочий стол**)
5. Вы получите папку `terra-club`

---

## ШАГ 3: Откройте папку в терминале

**На Windows:**
1. Откройте папку `terra-club` в Проводнике
2. Нажмите в адресной строке (где написан путь к папке)
3. Сотрите всё и напишите: `cmd`
4. Нажмите Enter — откроется чёрное окно (терминал)

**На Mac:**
1. Откройте Terminal (через поиск Spotlight: Cmd + Пробел → "terminal")
2. Напишите команду: `cd ~/Desktop/terra-club` и нажмите Enter
   (или перетащите папку `terra-club` в окно терминала после `cd `)

---

## ШАГ 4: Установите зависимости (один раз)

В терминале напишите точно эту команду и нажмите Enter:

```
npm install
```

⏳ Подождите 1-3 минуты. Вы увидите много текста — это нормально.
Когда всё готово, появится строка готовая к вводу.

---

## ШАГ 5: Запустите приложение

Напишите команду:

```
npm start
```

⏳ Подождите 20-40 секунд. Браузер откроется автоматически на странице:
**http://localhost:3000**

Если браузер не открылся сам — откройте его и введите этот адрес.

## Что умеет приложение (по ролям)

### 👥 Гость / Новый участник
- Читает новости и анонсы города
- Видит правила клуба
- НЕ может писать в большинстве чатов
- НЕ имеет доступа к школам и наставничеству

### 🎓 Ученик школы
- Всё что у Гостя + возможность писать в общих чатах города
- **Кабинет школы**: прогресс по модулям, задания, медиатека
- Общение с однокурсниками и преподавателем в чате школы
- Просмотр видео и документов своей школы

### 👨‍🏫 Преподаватель
- **Кабинет школы**: список всех учеников с прогрессом
- Создание заданий и дедлайнов
- Загрузка учебных материалов (видео, документы, аудио)
- Проверка домашних заданий с оценками
- Написать любому ученику напрямую

### 🌱 Наставляемый
- Общий чат своей группы наставничества
- Личный чат с наставником
- Видит других участников своей группы
- Сдаёт задания наставнику

### 🔮 Наставник
- Список всех своих наставляемых с прогрессом целей
- Создание и проверка заданий
- Личный чат с каждым наставляемым
- Заметки по каждому участнику
- История встреч

### ⚙️ Руководство (Администратор)
- **Дашборд**: статистика по всей сети (22 города, 6 стран)
- **CRM**: список всех участников с фильтрами по ролям и городам
- Все чаты всех городов
- Аналитика по школам и потокам
- Управление пользователями

## Разделы приложения

### 💬 Сообщения (Мессенджер)
Работает как Telegram, но всё в одном месте:
- **Общие группы города**: Новости, Основной чат, Нетворкинг, ДОБРО, Мастер-классы, Спорт, Правила, Дети
- **Кабинет школы**: Общий чат, Домашние задания, Материалы, Результаты
- **Наставничество**: Группа, Задания, Прогресс, Личный чат

### 🎓 Кабинет школы / Ученика
- Прогресс по модулям (с кольцом прогресса)
- Расписание офлайн-встреч
- Текущее задание с дедлайном
- Медиатека: видео, документы, аудио
- История оценок и обратная связь

### 🌱 Наставничество
- Для наставника: карточки каждого наставляемого с прогрессом
- Профиль, задания, заметки, история встреч по каждому
- Для наставляемого: карточка наставника, цель, задания, группа

### 👥 CRM (только для Руководства)
- Таблица всех участников
- Фильтры: по роли, городу, имени
- Карточка участника: контакты, школа, поток
- Быстрые действия: написать, редактировать

### 📊 Дашборд (только для Руководства)
- Цифры: 2847 участников, 18 школ, 184 пары наставничества
- Топ школ по количеству учеников
- Участники по городам с прогресс-барами
- Лента последних событий

## Частые вопросы

**Браузер не открывается сам?**
Просто откройте любой браузер и введите адрес: `http://localhost:3000`

**Пишет ошибку "command not found"?**
Значит Node.js не установился. Вернитесь к Шагу 1 и попробуйте снова.

**Пишет "port 3000 is already in use"?**
Откройте `http://localhost:3001` в браузере — приложение запустилось на другом порту.

**Хочу поменять данные пользователей?**
Откройте файл `src/data/db.js` в любом текстовом редакторе (Notepad, TextEdit) и измените нужные данные.

---

## Следующие шаги для продакшена

Когда приложение готово к реальному запуску в интернете:

1. **База данных**: Заменить `src/data/db.js` на реальный backend (Firebase, Supabase, или PostgreSQL)
2. **Хостинг**: Задеплоить на Vercel, Netlify или собственный сервер командой `npm run build`
3. **Мобильные приложения**: Создать React Native версию на основе того же кода
4. **Авторизация**: Подключить реальную систему регистрации (Firebase Auth, Auth0)
5. **Файлы**: Подключить облачное хранилище для медиафайлов (AWS S3, Cloudinary)
